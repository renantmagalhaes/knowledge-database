## Contributing

New content is picked up automatically on every push to `master` — just add a markdown file under the relevant topic folder in `docs/` (or create a new one) and it appears in the sidebar on the next deploy. See the [repository README](https://github.com/renantmagalhaes/knowledge-database#contribution) for details.
